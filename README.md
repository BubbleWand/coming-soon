# Bubble ```coming-soon```

A temporary landing page for information and email subscription sign up.

## Prerequisites

- [NodeJS](https://nodejs.org/en/download/)
- [npm](https://www.npmjs.com/get-npm)

## Installation

1. Download or clone `coming-soon`
2. Run the following commands

```bash
$ cd coming-soon

$ npm install

$ node app.js
```
